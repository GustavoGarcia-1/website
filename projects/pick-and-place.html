Pick and Place Robot

Overview

This project applies deep reinforcement learning to train a simulated robot arm for pick-and-place manipulation. Using MuJoCo MJX and Brax for GPU-accelerated physics simulation, I built a fully parallelized training pipeline in JAX that runs 2,048 environments simultaneously — enabling the agent to accumulate 160M+ environment steps efficiently.

The core challenge was reward engineering: designing a multi-phase shaped reward function that guides the agent through reaching, grasping, lifting, and placing — without creating reward hacking opportunities or local optima.

Environment Design

The environment was built on MuJoCo MJX, which compiles physics simulation to XLA for GPU execution via JAX. This allowed running 2,048 parallel rollouts on a single GPU, dramatically accelerating data collection compared to CPU-based simulation.

The task involves a robot arm that must pick up an object from a randomized starting position and place it at a target location. Observations include joint positions, velocities, end-effector pose, object pose, and target position. Actions control joint torques directly.

Reward Shaping

The reward function uses a multi-phase structure that guides the agent through the full pick-and-place sequence. Each phase activates based on task progress, preventing the agent from exploiting early-phase rewards while ignoring later objectives.

Key components include tanh-based distance shaping (providing smooth gradients at all distances), velocity-based stillness penalties (discouraging jerky motions during grasping), and grasp success bonuses that gate the transition to the placement phase.

# Multi-phase shaped reward function
def compute_reward(state):
    ee_pos = state.end_effector_pos
    obj_pos = state.object_pos
    target_pos = state.target_pos

    # Phase 1: Reach — tanh distance shaping
    reach_dist = jnp.linalg.norm(ee_pos - obj_pos)
    reach_reward = 1.0 - jnp.tanh(5.0 * reach_dist)

    # Phase 2: Grasp — stillness penalty + contact bonus
    velocity_penalty = -0.1 * jnp.linalg.norm(state.ee_velocity)
    grasp_bonus = 2.0 * state.is_grasped

    # Phase 3: Place — distance to target (gated on grasp)
    place_dist = jnp.linalg.norm(obj_pos - target_pos)
    place_reward = state.is_grasped * (1.0 - jnp.tanh(5.0 * place_dist))

    return reach_reward + velocity_penalty + grasp_bonus + place_reward

Training & Results

The agent was trained using PPO (Proximal Policy Optimization) for 160M+ environment steps. Training was tracked via Weights & Biases, logging episode returns, success rates, phase completion rates, and policy entropy over the course of training.

The parallelized JAX pipeline made it feasible to run large-scale hyperparameter sweeps over reward coefficients, learning rates, and network architectures. The final policy achieved consistent pick-and-place success across randomized object and target positions.
